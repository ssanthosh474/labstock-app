🛠 Auto-Sync Setup Guide (Free)
This one-time setup takes ~30 mins and costs ₹0/month using free tiers.
1
Deploy the included server.js to Railway.app or Render.com (free tier)
2
Create a Slack App at api.slack.com/apps → Enable Event Subscriptions
3
Set Request URL to your server: https://your-app.railway.app/slack/events
4
Subscribe to event: message.channels → Install app to your workspace
5
Paste your server URL above. Done! Every Slack message auto-updates stock.
